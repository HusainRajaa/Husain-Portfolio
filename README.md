# Modern Developer Portfolio

A high-performance, visually stunning personal portfolio website designed to showcase development skills and projects. Built with a focus on **absolute performance and design precision**.

## The Power of Pure Web Technologies

This project stands as a testament to the raw power of standard web technologies. **No frameworks. No build steps. No bloat.**

By utilizing **Pure HTML5, CSS3, and Vanilla JavaScript**, this portfolio delivers:

- **Instant Load Times**: Near-zero parsing overhead compared to React/Next.js bundles.
- **Butter-Smooth Animations**: 60fps transitions and scroll effects optimized directly for the browser's painting engine.
- **Pixel-Perfect Control**: every pixel and interaction is handcrafted for a premium feel without fighting framework constraints.
- **Universal Compatibility**: Works flawlessly across all modern browsers without hydration errors or complex dependency chains.

## Features

- **Premium Dark Theme**: Sleek, modern aesthetic with ambient glow effects.
- **Interactive Hero Section**: Full-height hero with parallax visuals and entry animations.
- **Dynamic Project Carousel**: Draggable horizontal slider for showcasing work.
- **Skills Filtering**: Instant client-side filtering system for technical skills.
- **Responsive Design**: Fluid layout that adapts perfectly from 4K desktops to mobile devices.
- **Intersection Animations**: Elements fade and slide in gracefully as you scroll.

## Tech Stack

- **HTML5**: Semantic, accessible markup structure.
- **CSS3**: Advanced usage of CSS Variables, Flexbox, Grid, and keyframe animations.
- **JavaScript (ES6+)**: Modular codebase handling UI interactions, scroll logic, and DOM manipulation.

## Content

The repository includes:

- `index.html` - The core structure.
- `styles.css` - The design system and responsive styles.
- `app.js` - The interactive logic.
